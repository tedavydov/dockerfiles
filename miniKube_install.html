1) узнать последнюю версию kubectl
https://storage.googleapis.com/kubernetes-release/release/stable.txt

скачать нужную версию kubectl
https://storage.googleapis.com/kubernetes-release/release/v1.22.0/bin/linux/amd64/kubectl

-----------------------------
curl -LO https://storage.googleapis.com/kubernetes-release/release/v1.22.0/bin/linux/amd64/kubectl
-----------------------------

Сделайте двоичный файл kubectl исполняемым:

chmod +x ./kubectl


2) скачать последнюю версию minikube

https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64 
-----------------------------
curl -Lo minikube https://storage.googleapis.com/minikube/releases/latest/minikube-linux-amd64 && chmod +x minikube
-----------------------------

chmod +x minikube
